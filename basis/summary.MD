**基础概念**

**并发**
```
问题根源
- 共享数据的隔离型
  - 单CPU通过线程同时切换读写数据
  - 多CPU同时读写数据
- 代码的有序性
  - 编译优化调整代码顺序
```

**锁**
```
双重判定锁
- 过程
  判断-加锁-判断
- 作用
  - 保证线程的安全性
  - 降低锁导致的性能损失
```
