# 数据结构 #
数据结构是数据的存储、组织结构。

# 数组 #
数组是用一块连续的内存空间来存储一组相同类型的数据。

特点：读取快，插入、删除慢

# 链表 #
占用随机内存；每个元素存贮下一个元素的地址；读取慢，插入，删除快

# 栈 #
先进后出，压入、弹出

# 队列 # 
先进先出，入队、出队

# 散列表 #
散列函数：把任意长度的输入变换成固定长度的输出
  - 较低的装填因子
  - 良好的散列函数
