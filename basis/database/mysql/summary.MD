# 逻辑架构 #
  - 客户端
  - SERVER
    - 连接器 建立连接，读取权限，校验权限
    - 分析器 词法分析，语法分析，语义分析
    - 优化器 选择索引，生成执行计划
    - 执行器 校验权限，调用引擎
  - 存储引擎 操作数据
