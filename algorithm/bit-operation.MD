# 使用 #
```
运算
- ^ 按位异或
- & 按位与
- | 按位或
- ~ 取反
- << 算术左移
- >> 算术右移

技巧
- n & (n - 1) 去掉最后一个1
- n & (-n) 得到最后一个1
```
