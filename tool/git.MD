# GIT #

## 定义 ##
版本控制系统

## 命令 ##
  - create & clone
    - git init
    - git clone path
  - add & remove
    - git add file
    - git add *
    - git rm file
  - commit & synchronize
    - git commit -m "msg"
    - git push origin
    - git remote add origin server
    - git pull origin
  - restore
    - git checkout head
    - git reset head
    - git checkout -- 
  - branch
    - git branch
    - git branch <branchname>
    - git checkout <branchname>
    - git checkout -b <branchname>
    - git branch -d <branchname>
    - git merge <branchname>
  - tag
    - git tag
    - git tag <tagname>
    - git tag -a <tagname> -m "msg"
    - git tag -d <tagname>
  - log
    - git log
    - git reflog
