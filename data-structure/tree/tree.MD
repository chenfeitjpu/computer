## 简介
```
每个节点有一个前驱节点，有多个后继节点，根节点没有前驱节点，叶子节点没有后继节点

分类
- 二叉树
  - 满二叉树  每一个层的结点数都达到最大值  
  - 完全二叉树  除最后一层外，每一个层的结点数都达到最大值，最后一层缺失的节点都在右侧
  - 二叉搜索树 
- 多叉树
  - 多叉搜索树
```

## 堆
```
定义
堆是一棵完全二叉树

分类
- 大顶堆 任意节点不小于左右子节点
- 小顶堆 任意节点不大与左右子节点
```
