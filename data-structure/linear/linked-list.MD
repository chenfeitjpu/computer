## 简介
```
分散存放元素，通过指针连接，查找元素通常要O(n)，已知位置时插入删除是O(1) 
```

## 双指针  
```
定义
- 快慢指针 两个指针同向而行，一快一慢  
- 左右指针 两个指针相向而行或者相背而行
```
