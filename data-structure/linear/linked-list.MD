# 定义 #
随机存储元素

# 特点 #
查询慢O(n)，增删快O(1)
  
# 应用 #
## 双指针 ##
**定义**  
- 左右指针 两个指针相向而行或者相背而行
- 快慢指针 两个指针同向而行，一快一慢

**应用**  
*LeetCode*  
- 83.删除排序链表中的重复元素
- 876.链表的中间结点
- 19.删除链表的倒数第N个结点
