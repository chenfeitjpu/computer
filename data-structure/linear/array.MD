# 定义 #
```
顺序存储元素，查询快O(1)，增删慢O(n)
```

# 使用 #
## 前缀和 ##  
```
原始数组不变，频繁查询某个区间的累加和  
```

应用场景
- 求一维数组区间值
- 求二维数组区间值
- 查找指定区间值

```
- 303.区域和检索-数组不可变
- 304.二维区域和检索-矩阵不可变
- 560.和为K的子数组
```
</details>
