# 定义 #
```
在内存中按顺序、连续地存放一组相同类型的元素
通过下标直接访问元素，时间复杂度是O(1)
在中间插入、删除元素时，需要搬动后面的元素，时间复杂度是O(n)
```

# 使用 #
## 前缀和 ## 
```
定义
从起始位置累加到当前位置的总和

应用场景
原始数组不变，频繁查询某个区间的累加和  
- 求一维数组区间值
- 求二维数组区间值
- 查找指定区间值
```

## 差分数组 ##
```
定义
记录当前元素和上一个元素的差值

应用场景
频繁对原始数组的某个区间的元素进行增减  
```
