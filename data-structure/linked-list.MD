# 定义 #
随机存储元素

# 特点 #
查询慢O(n)，增删快O(1)
  
# 应用 #
## 双指针 ##
*简介*  
- 左右指针 两个指针相向而行或者相背而行
- 快慢指针 两个指针同向而行，一快一慢
