# 定义 #
随机存储元素

# 特点 #
查询慢O(n)，增删快O(1)

# 方法 #
## 双指针 ##
*简介*  
- 左右指针 两个指针相向而行或者相背而行
- 快慢指针 两个指针同向而行，一快一慢

*应用*
LeetCode
- 83.删除排序链表中的重复元素（简单）
- 876.链表的中间结点（简单）
- 160.相交链表（简单）
- 19.删除链表的倒数第 N 个结点（中等）
