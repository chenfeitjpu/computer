# 定义 #
```
定义  
一个节点对应多个子节点  
分类
- 二叉树
  - 满二叉树  每一个层的结点数都达到最大值  
  - 完全二叉树  除最后一层外，每一个层的结点数都达到最大值，最后一层缺失的节点都在右侧
  - 二叉搜索树
    - 平衡二叉搜索树
      - AVL树
        - 概念
          - 具有二叉查找树的全部特性
          - 每个节点的左子树和右子树的高度差不超过1
    - 红黑树  
- 多叉树
  - 多叉搜索树
    - 2-3-4
```

## 二叉搜索树 ##
```
- 定义

  - 右子树节点的值都比根结点的值要大
- 应用

    - 230.BST第K小的元素
    - 538.二叉搜索树转化累加树

```

## 堆 ##
```
- 定义 
  堆是一棵完全二叉树  
  - 大顶堆 任意节点不小于左右子节点
  - 小顶堆 任意节点不大与左右子节点
- 应用  
  - LeetCode  
    - 23.合并K个升序链表  
```
