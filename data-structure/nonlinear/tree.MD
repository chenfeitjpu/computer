# 定义 #
```
定义  
一个节点对应多个子节点
特点  
一对多
分类
- 满二叉树
  每一个层的结点数都达到最大值  
- 完全二叉树
  除最后一层外，每一个层的结点数都达到最大值  
  最后一层有空缺结点且空缺在左边
- 二叉搜索树
  - 平衡二叉搜索树
    - 2-3-4
    - AVL树
      - 概念
        - 具有二叉查找树的全部特性
        - 每个节点的左子树和右子树的高度差至多等于1
      - 操作  
        - 左-左型  右旋
        - 右-右型  左旋
        - 右-左型  先右旋变成右-右型再左旋
        - 左-右型  先左旋变成左-左型再右旋
    - 红黑树      
```
