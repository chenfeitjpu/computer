## 简介
PHP-FPM的工作模式
- master 进程：管理 worker
- worker 进程：处理请求

一个PHP请求的生命周期
- Nginx接收请求
- 转发给PHP-FPM
- worker解析脚本
- 执行opcode
- 返回结果
- 释放内存

## 数据结构
```
zval
zval是PHP中变量的底层数据结构
- 结构
  - value（实际值）
  - type（类型）
  - refcount（引用计数）
  - is_ref（是否引用）

array
PHP数组底层是一个 HashTable（哈希表）
- 同时支持数字键和字符串键
- 保留插入顺序（PHP7+）
- 既能当数组，也能当map
```

## 内存管理
```
Copy On Write（写时拷贝）
- 多个变量指向同一个 zval
- 只有当某个变量被修改时才会复制
- 避免无意义内存拷贝，提高性能
```
