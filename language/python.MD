## 数据结构 ##
### 基础类型 ###
<details>
<summary>example</summary>

```
print(0b100)            # 二进制整数
print(0o100)            # 八进制整数
print(100)              # 十进制整数
print(0x100)            # 十六进制整数
print(123.456)          # 数学写法
print(1.23456e2)        # 科学计数法
print('hello world')    # 字符串
print(True)             # bool
print(False)            # bool
```
</details>
