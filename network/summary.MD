# 定义 #
```
在不同设备之间进行通信，实现资源共享和信息交换的系统
```

# 内容 #
```
网络分层  
- 应用层  HTTP、FTP、SMTP、DNS
- 传输层  TCP、UDP
- 网络层  IP、ICMP、IGMP
- 数据链路层  ARP、RARP
- 物理层  

网络模型
- OSI七层模型  理论上的通信模型
  应用层、表示层、会话层、传输层、网络层、数据链路层、物理层
- 五层模型  介绍原理使用的通信模型
  应用层、传输层、网络层、数据链路层、物理层
- TCP/IP模型  实际使用的通信模型
  应用层、传输层、网际层、网络接口层

服务端口
http(80)、https(443)、ftp(21)、mysql(3306)、redis(6379)、mongodb(27017)

浏览器中输入一个网址后，具体发生了什么
- DNS解析
- TCP三次握手
- HTTP请求发送
- 服务器处理请求
- 浏览器渲染页面
- TCP四次挥手
```
